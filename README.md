# Web Application Security Assessment Project

## Overview
This project demonstrates a full web application penetration testing lifecycle, simulating a real-world security consulting engagement.

## Targets
- OWASP Juice Shop
- DVWA (Damn Vulnerable Web Application)

## Methodology
1. Reconnaissance
2. Vulnerability Scanning (Nmap, Nikto)
3. Manual Exploitation
4. Risk Assessment (CVSS)
5. Remediation
6. Re-testing

## Tools Used
- Nmap
- Nikto
- Burp Suite
- Metasploit
- OWASP ZAP

## Deliverables
- Professional Penetration Test Report
- Risk Classification (CVSS Scoring)
- Mitigation Plan
- Secure Configuration Checklist

## Key Skills Demonstrated
- Web Application Security Testing
- Vulnerability Analysis
- Exploitation Techniques
- Security Hardening
- Technical Reporting

## Disclaimer
This project is conducted in a controlled lab environment for educational purposes only.
